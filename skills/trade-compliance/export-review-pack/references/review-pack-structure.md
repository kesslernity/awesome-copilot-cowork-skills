# Export Review Pack — Structure & Rules

The structure for the DRAFT export review pack. It prepares an export-control review; it never classifies, screens, clears, or decides a licence.

## Document structure

1. **Header / DRAFT line** — "DRAFT export review for <transaction> — generated <date>. Preparation only — not a classification, screening result, licence determination, or clearance. Trade compliance decides, against current official sources."
2. **Parties to screen** — table + checklist (run in the screening tool).
3. **Classification worksheet** — item facts + candidate categories to verify (no ECCN/USML/dual-use assigned).
4. **Red-flag review** — standard indicators, each Present / Not evident / Unclear with detail.
5. **Licence-determination questions** — the facts/questions that drive licence need (no conclusion).
6. **Open items** — missing info and next steps.

## Parties-to-screen table

| Party | Role | Full name | Address / country | Missing details |
|---|---|---|---|---|

Roles to capture: buyer, end-user, consignee, freight forwarder, bank, agents, intermediaries, ultimate parent (if stated). Never mark a party as a match or "clear".

## Red-flag indicators to check

Reluctance to share end-use; capability mismatch; unusual routing/transshipment; freight forwarder as end-user; declined installation/training; unusual payment (cash/third-party/over-payment); evasive answers; destination/end-user inconsistent with the order. Report each Present / Not evident / Unclear with the supporting detail and what to obtain.

## Never include

- A classification (ECCN, USML/ITAR category, dual-use, EAR99) — only "verify whether".
- A screening match or a clearance of any party.
- A licence-required / not-required conclusion, or an exception applied.
- A decision to ship, hold, or proceed.

Screening runs in the screening tool against current official lists. Classification, licensing, and clearance are the export control officer's — against current official sources, with civil and criminal liability attached.
