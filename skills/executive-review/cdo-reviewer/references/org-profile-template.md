# Organisation Profile (template)

This file gives reviewer skills context about YOUR organisation, so persona findings land on your business instead of a generic one. To use it: copy this file, fill in every section, and save it to your OneDrive as `/Documents/Cowork/org-profile.md`. Reviewer skills load that path automatically at the start of every review.

Rule for the reviewing agent when this template is used unfilled: every item below is UNKNOWN. Do not assume an answer to any of them. Where a persona finding depends on one of these unknowns, state the dependency as an open question instead of guessing.

## Business model

How the organisation earns money: what is sold, to whom, on what pricing model (subscription, licence, project, transaction), and roughly what share of revenue each line carries.

Default if unfilled: unknown.

## Customers and segments

Who buys: segments, typical customer size, the top customer concentration if any, and which segments are growing or shrinking.

Default if unfilled: unknown.

## Stage and size

Headcount, revenue band, funding or ownership situation (bootstrapped, VC-backed, listed, family-owned, public sector), and growth stage.

Default if unfilled: unknown.

## Current priorities

The three to five stated priorities for the current planning period, in the organisation's own words. Personas use these to test strategic fit.

Default if unfilled: unknown.

## Constraints

Hard limits the board must respect: budget ceilings, hiring freezes, regulatory regimes, technology standards, contractual commitments, geographic restrictions.

Default if unfilled: unknown.

## Recent strategic decisions

Decisions already taken in the last 12 months that new proposals must be consistent with (or explicitly argue against): platform choices, market exits, reorganisations, major vendor commitments.

Default if unfilled: unknown.

## Sensitivities and no-go areas

Topics where the organisation has been burned before, lines it will not cross, and anything a reviewer should treat with extra scepticism or extra care.

Default if unfilled: unknown.
